Premium, conversion-focused landing page for a beauty salon built with pure HTML, CSS, and vanilla JavaScript.

The goal of this project is to demonstrate frontend implementation quality for commercial marketing pages without frameworks.

## Live Demo

- Repository: https://github.com/Smurf228/beauty-salon-landing-page-portfolio
- GitHub Pages: https://smurf228.github.io/beauty-salon-landing-page-portfolio/

## Case Summary

### Objective

Build a high-end one-page salon website that converts traffic into bookings while keeping implementation lightweight and maintainable.

### Solution

- Hero-first layout with strong CTA hierarchy
- Trust-focused storytelling and service framing
- Gallery with lightbox interaction
- Testimonials carousel in vanilla JavaScript
- Minimal booking form with client-side validation
- Fully responsive mobile-first behavior

### Result

- Portfolio-ready landing page with realistic business structure
- Production-style UI interactions without external libraries
- Clear showcase of semantic HTML, scalable CSS architecture, and practical JavaScript

## Features

- Sticky header with smooth in-page navigation
- Premium hero section with dual CTA actions
- Services grid with hover states
- About section with trust-oriented copy
- Portfolio gallery with image lightbox
- Testimonials slider (vanilla JS)
- Booking form with inline validation
- Final conversion block and contact footer

## Tech Stack

- HTML5 (semantic sections)
- CSS3 (custom design system, Grid, Flexbox, transitions)
- Vanilla JavaScript (UI behavior and form validation)

## Run Locally

1. Clone the repository.
2. Open the project folder.
3. Launch `index.html` in your browser.

No build step and no dependencies are required.
