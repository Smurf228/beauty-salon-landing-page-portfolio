# Lumea Studio Landing Page

Premium, conversion-focused landing page for a beauty salon built with pure HTML, CSS, and vanilla JavaScript.

This project demonstrates frontend fundamentals for commercial marketing pages without frameworks.

## Live Project Scope

- Sticky header with smooth in-page navigation
- Premium hero section with clear CTAs
- Services grid with hover states
- About section with trust-oriented storytelling
- Gallery with lightbox preview
- Testimonials slider (vanilla JS)
- Booking form with client-side validation
- Final call-to-action section
- Responsive layout for desktop and mobile

## Tech Stack

- HTML5 (semantic sections)
- CSS3 (custom design system, Grid, Flexbox, transitions)
- Vanilla JavaScript (UI behavior and form validation)

## Project Structure

.

## How To Run Locally

1. Clone the repository.
2. Open the project folder.
3. Launch `index.html` in your browser.

No build step and no dependencies are required.
